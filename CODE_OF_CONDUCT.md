# Code of Conduct

## Our Commitment

The Earthcast MCP project is committed to providing a professional, respectful, and collaborative environment for all contributors, users, and community members. We value technical excellence, constructive dialogue, and mutual respect.

## Professional Standards

We expect all participants to:

- **Communicate professionally** - Use clear, respectful language in all interactions
- **Focus on technical merit** - Evaluate contributions based on their technical quality and alignment with project goals
- **Provide constructive feedback** - Offer specific, actionable suggestions when reviewing code or proposals
- **Respect diverse perspectives** - Acknowledge that contributors may have different approaches, backgrounds, and expertise levels
- **Be responsive and collaborative** - Engage in good faith discussions and work together to solve problems
- **Credit contributions appropriately** - Acknowledge the work and ideas of others

## Unacceptable Behavior

The following behaviors are not acceptable:

- Harassment, intimidation, or discriminatory language
- Personal attacks or inflammatory comments
- Publishing others' private information without permission
- Trolling, deliberate disruption, or off-topic arguments
- Any conduct that could be reasonably considered inappropriate in a professional setting

## Scope

This Code of Conduct applies to:

- All project repositories and issue trackers
- Pull requests, code reviews, and discussions
- Project documentation and communications
- Any public space where someone is representing the project

## Reporting Issues

### Technical Issues
For bugs, feature requests, and technical questions, please use [GitHub Issues](https://github.com/ec-will/earthcast-mcp/issues).

### Security Vulnerabilities
For security-related concerns, please follow our [Security Policy](SECURITY.md).

### Code of Conduct Violations
For conduct-related concerns, please contact the project maintainers at the repository. All reports will be reviewed and investigated promptly and confidentially.

## Enforcement

Project maintainers have the right and responsibility to:

- Remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct
- Temporarily or permanently ban any contributor for behaviors they deem inappropriate, threatening, or harmful
- Take any other action deemed necessary to maintain a professional environment

## Attribution

This Code of Conduct is adapted for professional technical collaboration in the Earthcast MCP project, which is built on the weather-mcp foundation by the Weather MCP Server Contributors.

## Questions

If you have questions about this Code of Conduct, please open an issue in the repository for discussion.
