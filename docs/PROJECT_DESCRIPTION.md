# PROJECT DESCRIPTION

This project will be an MCP server that can be used by any AI system to access weather data from NOAA's API. The focus should be on any AI system but mainly for use with Claude Code locally.

This project will be posted to github once working and tested.

# REQUIREMENTS

- The MCP server should be able to get the local forcast for any location
- The MCP server should be able to get historical weather data for any location for a day, week, month, or custom date range.
